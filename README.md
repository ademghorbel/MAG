# Mohamed Adam Ghorbel — Portfolio

A high-performance, design-forward portfolio website showcasing my technical projects, graphic design work, and community volunteering.

## 🎯 Overview
This repository contains the source code for my personal portfolio. It is built to be fast, responsive, and visually engaging, reflecting my dual background in:
- **Data Engineering & AI Systems**
- **Graphic Design & Visual Communication**

## ✨ Features
- **Data-Driven Architecture:** All content is decoupled from the HTML and managed via a central `data.js` file, acting as a lightweight CMS.
- **Optimized Performance:** Fully optimized with WebP images and Google Drive video streaming to keep the repository lightweight and fast.
- **GSAP Animations:** Smooth page transitions, parallax scrolling, magnetic buttons, and custom cursor effects powered by GreenSock.
- **Zero Framework Bloat:** Built with Vanilla HTML, CSS, and JavaScript for maximum speed and control.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Properties, Grid/Flexbox)
- **Logic:** Vanilla JavaScript (MVC pattern)
- **Animations:** GSAP 3 & ScrollTrigger
- **Media:** WebP images, Iframe video embedding

## 📁 Repository Structure
- `index.html`, `work.html`, `projects.html` — Core pages.
- `/assets/css/` — Global styling and design tokens.
- `/assets/js/` — Controllers, animation engine, and data model.
- `/events/` — Detailed case studies for individual events.

The portfolio is hosted via GitHub Pages and is automatically updated when pushed.
